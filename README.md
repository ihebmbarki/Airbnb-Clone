# Airbnb Clone

A SwiftUI-based clone of the Airbnb app, built as a learning project to explore advanced SwiftUI concepts, modern UI design patterns, and integration with external APIs.

## Features Implemented

### Core Features
- **Property Listings:** Display a list of properties with key details such as name, location, price, and image.
- **Property Details:** Detailed view of each property, including description, amenities, and availability.
- **Search Functionality:** Users can search for properties based on criteria like location and price range.
